# Content Rules

Every lesson should include:

* Hero
* Objectives
* Lesson
* Examples
* Images
* Videos
* Quiz
* Common mistakes
* Takeaways
* FAQs

Lessons should teach beginners clearly.

Avoid unnecessary jargon.

Always recommend related lessons.

Integrate new lessons into:

* Search
* Recommendations
* Progress
* XP
* Bookmarks
* Recent Activity
