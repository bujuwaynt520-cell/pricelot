# PriceLot Brand Guide

# Brand Identity

PriceLot is a premium trading education platform.

The brand should communicate:

* Professionalism
* Trust
* Precision
* Discipline
* Growth
* Simplicity

Users should immediately feel that PriceLot is built by experienced traders and educators.

---

# Brand Personality

PriceLot is:

* Confident
* Professional
* Educational
* Calm
* Modern
* Honest

PriceLot is NOT:

* Flashy
* Scammy
* Overhyped
* Loud
* Gambling-focused

---

# Design Philosophy

Every page should prioritize:

* Readability
* Simplicity
* Performance
* Clean spacing
* Clear hierarchy

Avoid unnecessary decorations.

Whitespace is valuable.

---

# Primary Colors

Primary

Orange

Used for:

* Buttons
* Highlights
* Progress
* Links
* Active navigation

Secondary

Black

Used for:

* Titles
* Headers
* Navigation

Neutral

White

Backgrounds

Gray

Supporting UI

---

# Typography

Headers:

Large

Bold

Confident

Body:

Easy to read

Comfortable spacing

Avoid long unreadable paragraphs.

---

# UI Principles

Cards

Rounded corners

Soft shadows

Consistent spacing

Buttons

Large enough to click comfortably.

Clear hover states.

Inputs

Simple.

Clean.

Accessible.

---

# Icons

Use Lucide icons whenever possible.

Maintain a consistent icon style throughout the platform.

---

# Images

Images should:

* Teach
* Explain
* Support learning

Avoid decorative stock photos unless they genuinely improve understanding.

Prefer:

* Charts
* Diagrams
* Illustrations
* Trading screenshots
* Educational graphics

---

# Videos

Videos should:

* Be educational
* Be concise
* Support lessons
* Match the lesson content

---

# Writing Style

Use:

* Simple English
* Short sentences
* Clear explanations
* Real examples

Avoid:

* Buzzwords
* Fake promises
* Clickbait
* Excessive jargon

---

# User Experience

Users should always know:

* Where they are
* What they just learned
* What comes next

Learning should feel structured.

---

# Consistency

Every new page should feel like it belongs to PriceLot.

No feature should introduce a completely different design language.

Consistency is more important than novelty.
